Detailed Description

This page provides links to all of the C++ utilities which are available in libGRVY (organized by functionality). To access these routines from C++, make sure to #include <grvy.h> within your source code files. Unless specified otherwise, functions which have an integer return value return "0" upon success (and the primary exception to this rule is with a subset of the Input Parsing routines).
